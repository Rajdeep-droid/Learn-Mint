## 3. The Web Frontend (`D:\Mint-Token\frontend`)

### Core Files Created:
*   **`src/app/globals.css`**: The complete design system featuring dark themes, glowing accents (cyan/purple), micro-animations (confetti, pulse, fade-in), and custom scrollbars.

*   **`NavBar.tsx`**: The top navigation bar featuring your logo and the "Connect Wallet" button. It displays your wallet address and XLM balance.
*   **`VideoPlayer.tsx`**: A custom 9:16 vertical video player with a lock screen that requires the user to click "Unlock Course" before watching.
*   **`QuizModule.tsx`**: The interactive multiple-choice quiz. It animates shaking on a wrong answer and drops confetti when you pass.
*   **`CourseView.tsx`**: The master component that glues the video player and quiz together into one seamless flow.
*   **`RecentGraduates.tsx`**: A sidebar feed designed to listen to the blockchain and display real-time updates when other people pass the quiz.

### Wallet & Blockchain Integration:
*   **`src/context/WalletProvider.tsx`**: Manages the connection to the Freighter browser extension.
*   **`src/lib/stellar.ts`**: Contains all the helper functions for talking to the Stellar Testnet using Soroban RPC.
*   **`src/lib/eventListener.ts`**: The polling system to read live events from the blockchain.
